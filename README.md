# Herramienta de análisis deportivo

## Descripción del proyecto

Este proyecto consiste en el desarrollo de una herramienta de análisis deportivo enfocada a los clubes con escasos recursos económicos. Será una funcionalidad añadida a un proyecto previo de una red social dirigida a clubes y jugadores amateur. 
Las características del proyecto permitirán hacer análisis de forma exhaustiva tanto de clubes como de jugadores. Además, permitirá predecir el impacto que pueda tener un jugador a la hora de incluirlo en un equipo, predecir carencias y fortalezas de un equipo, o valorar el enfrentamiento contra otros clubes. 
De esta forma, se proporcionará a los clubes una herramienta que les acercará a la profesionalización, con el objetivo de minimizar las diferencias competitivas que derivan de las diferencias económicas. 

## Objetivos principales

El objetivo fundamental es el desarrollo completo de la herramienta de análisis. Para ello, se definen pequeños objetivos que ayudarán al progreso.

1. Herramienta de análisis de jugadores. El objetivo será valorar tanto sus capacidades como su valor en el mercado. Se desarrollarán distintos gráficos interactivos que permitan elegir las estadísticas que se quieran incluir
2. Herramienta de análisis de equipos. Se valorará el rendimiento de los distintos equipos, ya sea de cara a potenciar el propio equipo o preparar futuros enfrentamientos
3. Herramienta de predicción. Se incluirá a un jugador dentro de un equipo o a un equipo dentro de un conjunto para predecir el impacto del jugador dentro del equipo o el resultado final de un partido o una competición

## Plan de trabajo

1. Obtención y limpieza de datos
2. Creación de visualizaciones para el análisis
3. Creación de modelos de ML para las predicciones
4. Comprobación de los modelos
5. Creación de posibles modelos de ML para clasificación
6. Comprobación de los modelos
